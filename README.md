# Deployment of Smart Budget AI Website on AWS S3

`AWS Cloud Website Deployment`
 
### Objective
The objective of this assignment is to deploy a live website on AWS Cloud using Amazon S3 static website hosting. The project demonstrates basic cloud computing concepts, website hosting, and public deployment using AWS services.

### Project Description
The deployed project is a Smart Budget AI website that provides users with an interactive interface for budget analysis and expense management. The frontend was developed using HTML, CSS, and JavaScript.
Technologies Used
• HTML
• CSS
• JavaScript
• AWS S3
• Flask (Backend - local project)
• Python

### Deployment Steps
1. Created an AWS account and logged into AWS Management Console.
2. Opened Amazon S3 service.
3. Created a new S3 bucket with a unique bucket name.
4. Disabled block public access settings.
5. Uploaded the website files into the S3 bucket.
6. Enabled static website hosting from bucket properties.
7. Added a bucket policy to allow public read access.
8. Accessed the generated website endpoint URL to verify deployment.
   
### Challenges Faced
Understanding S3 bucket permissions and configuring bucket policies for public access were the major challenges during deployment.

### Outcome
The Smart Budget AI website was successfully deployed on AWS S3 and made publicly accessible using a live website URL.

### Conclusion
This assignment helped in understanding the basics of AWS Cloud services, static website hosting, bucket permissions, and real-world deployment processes.
